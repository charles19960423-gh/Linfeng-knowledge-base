## 核心理念

### 底层逻辑

认知以**「认知能力五要素」**为核心框架：

| 维度 | 本质属性 | 核心功能 |
|:---:|:---:|:---:|
| **思维模式** | 思维方式 | 塑造认知框架 |
| **学习能力** | 知识获取 | 提升学习效率 |
| **记忆方法** | 信息存储 | 增强记忆效果 |
| **问题解决** | 逻辑思维 | 提升解决能力 |
| **创新思维** | 创造性思考 | 激发创新能力 |

### 核心概念

1. **思维模式**：思维的方式与框架
2. **学习能力**：获取和吸收知识的能力
3. **记忆方法**：有效存储和提取信息的方法
4. **问题解决**：分析和解决问题的能力
5. **创新思维**：产生新想法和解决方案的能力

### 研究框架

```
认知
├── 思维模式
│   ├── 逻辑思维

└── 创新思维
    ├── 创造性思考
    ├── 发散思维
    └── 整合创新
```
