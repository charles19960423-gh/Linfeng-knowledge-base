# 《林峰系统论 · 人模块》总览

## 模块定位

### 层级位置

| 上级 | 本级 | 下级 |
|:---|:---|:---|
| 林峰系统论 | **人模块** | 认知/情感/行为/关系/成长/价值/潜能 |

### 核心内涵

研究人的本质、认知、情感、行为与发展

### 模块使命

人模块聚焦个体与群体的全面发展，解析认知模式、情感管理、行为决策、关系网络、成长路径、价值体系、潜能开发，构建人的完整认知框架。

---

## 核心理念

### 底层逻辑

人模块以**「人发展七维度模型」**为核心框架：

| 维度 | 本质属性 | 核心功能 |
|:---:|:---:|:---:|
| **认知** | 思维能力 | 构建认知体系 |
| **情感** | 情绪管理 | 调节情感状态 |
| **行为** | 行动表现 | 规范行为模式 |
| **关系** | 社交互动 | 建立关系网络 |
| **成长** | 发展进步 | 实现自我成长 |
| **价值** | 意义追求 | 形成价值体系 |
| **潜能** | 能力开发 | 挖掘内在潜能 |

### 核心概念

1. **认知**：感知、思维、学习、记忆等心理过程
2. **情感**：情绪、情感、心态、心理健康等
3. **行为**：行动、习惯、决策、执行力等
4. **关系**：人际、家庭、社会、职场等关系
5. **成长**：学习、发展、突破、自我提升等
6. **价值**：价值观、意义、目标、信念等
7. **潜能**：天赋、能力、创造力、潜力等
