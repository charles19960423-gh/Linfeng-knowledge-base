## 核心分析框架

### 底层逻辑

社会规范研究维持社会秩序的规则体系，核心在于**法律-道德-习俗的协同**：

| 规范类型 | 内涵 | 约束机制 |
|:---:|:---:|:---:|
| **法律规范** | 国家法律/法规 | 国家强制力 |
| **道德规范** | 社会公德/职业道德 | 舆论/良心 |
| **习俗规范** | 习惯/风俗/礼仪 | 传统/社群压力 |

### 主要矛盾

1. **刚性与柔性**：法律强制与道德教化
2. **传统与现代**：习俗变迁与制度创新
3. **普遍与特殊**：国家法律与地方规范

### 规范协同体系

```
法律为底线：违法行为必须追究
    │
道德为中线：失德行为舆论谴责
    │
习俗为上线：优良风俗社会倡导
```
