## 核心分析框架

### 底层逻辑

政策法规研究政策过程的运行规律，核心在于**问题-方案-执行-评估的循环**：

| 政策阶段 | 内涵 | 核心活动 |
|:---:|:---:|:---:|
| **议程设置** | 问题进入政策视野 | 问题识别、利益表达 |
| **方案制定** | 备选方案形成 | 方案设计、可行性分析 |
| **决策执行** | 政策落地实施 | 组织协调、资源配置 |
| **效果评估** | 政策效果反馈 | 监测评价、调整优化 |

### 主要矛盾

1. **科学性与民主性**：专家理性与公众参与
2. **全局与局部**：整体利益与地方利益
3. **稳定与灵活**：政策连续性与适应性

### 政策过程模型

```
问题识别 ──→ 议程设置 ──→ 方案制定
    ↑                            │
    └──── 效果评估 ◀── 执行监督 ◀─┘
```
