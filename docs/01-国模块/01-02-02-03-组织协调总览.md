## 核心分析框架

### 底层逻辑

组织协调研究跨部门协作的运行机制，核心在于**沟通-协作-整合**：

| 协调维度 | 内涵 | 关键机制 |
|:---:|:---:|:---:|
| **纵向协调** | 上下级衔接 | 汇报-指示-反馈 |
| **横向协调** | 同级部门配合 | 会商-分工-配合 |
| **外部协调** | 政府与企业社会 | 沟通-合作-共治 |

### 主要矛盾

1. **分工与合作**：部门边界与协同需求
2. **本位与大局**：部门利益与整体利益
3. **沟通与效率**：充分沟通与决策效率

### 协调机制类型

```
正式协调：会议/文件/程序
    │
非正式协调：沟通/协商/人情
    │
制度化协调：联席/会商/专班
```
