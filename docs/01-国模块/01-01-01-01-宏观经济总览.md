## 核心分析框架

### 底层逻辑

宏观经济研究国家层面的经济总量运行规律，核心在于**总供给-总需求平衡**：

| 核心变量 | 内涵 | 调控工具 |
|:---:|:---:|:---:|
| **GDP** | 经济总量 | 财政政策、货币政策 |
| **通货膨胀率** | 价格水平 | 利率、货币供应量 |
| **失业率** | 就业状况 | 产业政策、就业政策 |
| **国际收支** | 对外经济 | 汇率政策、贸易政策 |

### 主要矛盾

1. **增长与稳定**：经济增速与通胀/失业的平衡
2. **效率与公平**：市场效率与收入分配的协调
3. **内部与外部**：国内经济与对外经济的均衡

### 关键变量关系

```
货币政策 ─────┬──────→ 利率 ─────→ 投资/消费
              │
              ├──────→ 货币供应量 ───→ 通胀
              │
              └──────→ 汇率 ─────→ 出口/进口

财政政策 ─────┬──────→ 政府支出 ───→ 总需求
              │
              ├──────→ 税收 ─────→ 可支配收入
              │
              └──────→ 赤字/盈余 ───→ 经济周期调控
```

## 研究要点

### 核心问题清单

- [ ] 经济周期的成因与调控机制
- [ ] 财政政策与货币政策的协调配合
- [ ] 经济增长动能的转换与培育
- [ ] 通货膨胀与通货紧缩的治理

### 延伸研究方向

1. **新常态下经济转型**：从高速增长转向高质量发展
2. **数字经济与宏观经济**：数据要素对经济总量核算的影响
3. **全球化与去全球化**：国际分工重构下的宏观经济策略
4. **系统性金融风险**：宏观审慎监管框架
