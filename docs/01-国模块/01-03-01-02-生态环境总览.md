## 核心分析框架

### 底层逻辑

生态环境研究人与自然的关系，核心在于**污染治理-生态保护-生态修复**：

| 环境要素 | 内涵 | 核心问题 |
|:---:|:---:|:---:|
| **大气环境** | 空气质量/气候变化 | 雾霾/碳排放 |
| **水环境** | 水体质量/水生态 | 污染/富营养化 |
| **土壤环境** | 土壤质量/食品安全 | 重金属/面源污染 |
| **生态系统** | 森林/湿地/海洋 | 生态退化 |

### 主要矛盾

1. **发展与保护**：经济增长与环境保护
2. **局部与整体**：区域利益与全局环境
3. **当前与长远**：当代发展与代际公平

### 环境治理体系

```
源头控制：清洁生产/产业升级
    │
过程治理：污水处理/垃圾分类
    │
末端减排：达标排放/总量控制
```
